# Secure-Office-Network-Lab
Built and secured a professional SOHO network infrastructure with a dedicated web-based HR system, focusing on network hardening and traffic isolation

Key Features:

1. Secure Segmentation: Implemented VLANs with Router-on-a-Stick for controlled inter-departmental routing.

2. Device Hardening: Secured Cisco devices using Hashed Secret Passwords, Console protection, and SSH for encrypted management.

3. Strict Access Control: Engineered Extended ACLs to limit HR traffic to essential services only (SMTP, POP3, HTTP/S).

4. Layer 2 Defense: Deployed Port Security to mitigate unauthorized hardware access.

5. Internal Services: Configured DNS, Email-server, and a custom Web-based HR System for internal operations.

6. Legal Compliance: Integrated warning Banners (MOTD) on all Network Devices .
